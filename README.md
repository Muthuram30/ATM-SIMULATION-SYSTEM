# ATM-SIMULATION-SYSTEM
How It Works
<br>
User enters card (simulated by input)
<br>
Enters 4-digit PIN
<br>
Menu appears:
<br>
Check balance
<br>
Withdraw money
<br>
Deposit money
<br>
View mini statement
<br>
Exit
<br>

Each operation updates simulated account data
