# ATM-SIMULATION-SYSTEM
How It Works
<br>
User enters card (simulated by input)
<br>
Enters 4-digit PIN
<br>
Menu appears:
<br>
Check balance
Withdraw money
Deposit money
View mini statement
Exit

Each operation updates simulated account data
